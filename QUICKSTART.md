# Quick Start Guide

## 1. Install Backend Dependencies
```bash
cd backend
npm install
```

## 2. Install Frontend Dependencies
```bash
cd frontend
npm install
```

## 3. Start Backend Server
```bash
cd backend
npm start
```
Server runs on: http://localhost:5000

## 4. Start Frontend Server (in new terminal)
```bash
cd frontend
npm start
```
App runs on: http://localhost:3000

## 5. Access the Application
- Open http://localhost:3000 in your browser
- Register a new account
- Login with your credentials
- Click "Check Balance" to see your account balance with celebration!

## What's Included

✅ **User Registration** - Create account with ₹100,000 initial balance
✅ **JWT Authentication** - Secure token-based authentication
✅ **Balance Check** - Fetch balance with token verification
✅ **Celebration Animation** - Confetti effect when balance is displayed
✅ **Responsive Design** - Works on all devices

## Database

Automatically connected to Aiven MySQL with:
- KodUser table (user accounts)
- UserToken table (token storage)

Simply start the backend and the tables will be created automatically!

Enjoy your Kodbank experience! 🚀
